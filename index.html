Incident Report: Card Access Breach

Attention Students of Basisschool Sint-Augstinus. Our school portal security has been bypassed. The principal's AI-powered access mechanism, which uses visual recognition to scan staff ID cards, was the primary target.

Your Task: Replicate the authentication mechanism and reconstruct the security protocols to help us regain control. Do this by training a Teachable Machine model that can differentiate and recognize staff and principal cards. For this you need to first generate more randomized cards, by using Google Gemini. These cards can then be used to train the Teachable Machine model.

If it works, show the lead lecturer to gain access to the school portal and search for the mailbox and phishing mail that got to the principal.
